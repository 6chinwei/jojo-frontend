# jojo-frontend

## Prerequisites
 - [Node.js] runtime 
   - npm
 - [Gulp]
```sh
$ npm install -g gulp
```
 - [Bower]
```sh
$ npm install -g bower
```

## Usage
### [In project root folder]

 - install 3rd-party tools and library first.

```sh
$ npm install
$ bower install
```

 - local testing server.

```sh
$ gulp serve
```
